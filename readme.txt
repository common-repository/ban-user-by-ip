== Description ==

Ban User By IP is a simple plugin that allows you to ban who you want by inserting the user IP Address.

For more information, check out <a href="http://www.danycode.com/ban-user-by-ip/">Ban User By IP</a>

Plugin Features:

- Easy to use
- Add and Delete Banned IP

== Installation ==

1. Upload the Ban User By IP plugin to your blog and activate it. 
2. Add and Delete Banned IP through: Setting -> Ban User By IP
